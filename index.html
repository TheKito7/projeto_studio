Sobre o Projeto

Uma nova identidade para um espaço existente

Este projeto apresenta a reforma de uma residência de dois pavimentos com o objetivo de transformá-la em um estúdio fotográfico profissional.

A proposta busca integrar funcionalidade, conforto, iluminação e estética contemporânea, criando ambientes preparados para produções fotográficas, atendimento aos clientes e áreas administrativas.

Conceito Arquitetônico

Minimalismo Industrial

O projeto utiliza elementos industriais combinados com madeira, concreto aparente, iluminação indireta e grandes aberturas para proporcionar ambientes sofisticados e funcionais.
